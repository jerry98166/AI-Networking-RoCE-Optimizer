│  │

│     ├─ 優先編碼器邏輯
│     │  ├─ if (count[0] > 0) 輸出優先級0
│     │  ├─ else if (count[1] > 0) 輸出優先級1
│     │  ├─ else if (count[2] > 0) 輸出優先級2
│     │  └─ else 輸出優先級3
│     │

│     ├─ 更新讀指針和計數器
│     ├─ 生成m_axis_tvalid和m_axis_tdata
